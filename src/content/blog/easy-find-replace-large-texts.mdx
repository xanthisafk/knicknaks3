## Basic Find and Replace

At its simplest, Find & Replace allows you to swap one string of text for another.

**Example:**

- **Find:** `Old Company Name`
- **Replace:** `New Corporate Identity`

With a single click, every mention of the old name across thousands of lines is updated instantly.

### When to Use Basic Replacement

- Renaming variables in a script.
- Fixing a recurring typo in a long article.
- Updating dates or years in a document.

## Leveling Up with Regular Expressions (Regex)

Simple text replacement is great, but what if your pattern is more complex? What if you want to find all email addresses and replace them with `[HIDDEN]`? This is where **Regex** comes in.

Regex (Regular Expressions) allows you to search for _patterns_ rather than exact strings.

### Common Regex Examples:

- **Find numbers:** `\d+`
- **Find capitalized words:** `[A-Z][a-z]+`
- **Find anything in brackets:** `\[.*?\]`

Our tool supports full JavaScript Regex, allowing you to perform surgical edits on your data that would be impossible with standard search tools.

<Callout type="tip">
  **Advanced Tip:** Use **Capture Groups**! If you search for `(Item) (\d+)` and
  replace with `$2: $1`, you can swap the order of words and numbers throughout
  your entire document.
</Callout>

## Common Use Cases

### Data Cleaning

If you have a exported list where every line starts with a date you don't need, you can use a Regex pattern like `^\d{4}-\d{2}-\d{2}\s` and replace it with an empty string to strip them all away in one go.

### Formatting Logs

Quickly find and remove timestamps or log levels (like `[INFO]`, `[ERROR]`) to make logs more readable for human review.

### Coding Refactors

Change a repeated function call or property name across multiple blocks of code when you don't have your full IDE open.
